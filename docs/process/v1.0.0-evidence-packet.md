## Boundary Statement

`v1.0.0` provides an externalized governance system: state, evidence, approvals, runtime measurements, provider evidence summaries, and release gates are inspectable outside the model.

It does not provide provider-native persistent session control. Runtime measurements are local/runtime facts; token and cost remain placeholders unless a provider reports them directly.
