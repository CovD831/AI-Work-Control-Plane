# v1.0.0

AI-Work-Control-Plane `v1.0.0` is the first sealed local-first AI Work Control Plane release.

## Status

Release sealed for local long-cycle agent work.

## What Is Included

- Governed plan sessions with persisted context, review, approval, execution, and evidence artifacts.
- Workspace status, context packets, topology snapshots, approval queues, evidence gates, runtime inspection, and recovery guidance as control-plane surfaces.
- Runtime measurement for local command jobs, including duration, exit status, provider/runtime metadata, degraded reasons, and operation receipts.
- Codex Runtime Pilot evidence consumption through provider evidence summaries without claiming provider session ownership.
- Portable governance bundle export and offline inspection through `team governance-bundle export` and `team governance-bundle inspect`.

## Delta From rc.3

- Promotes the package version from `1.0.0rc3` to `1.0.0`.
- Adds the final release notes and evidence packet.
- Updates README and release checklist to use the final release label.
- Carries forward the post-rc.3 externalization hardening proof: governance bundle export/inspect is complete, auditable, and non-blocking.
