# v1.0.0-rc.1

Agent Orchestrator `v1.0.0-rc.1` is the first release candidate for evaluating the project as a local-first AI Work Control Plane.

## Status

Runtime Measurement Ready for evaluation.

## Highlights

- Reframes the project around the AI Work Control Plane: workspace state, context packets, strategy decisions, topology snapshots, approvals, evidence, memory provenance, and recovery.
- Keeps the existing orchestration runtime as a lower execution layer instead of the product identity.
- Exposes runtime measurement readiness through `team setup --runtime command --format json`.
- Adds runtime measurement metrics and deltas to evidence reporting.
- Distinguishes measured, placeholder, and unavailable runtime facts.
- Reports release readiness through setup, compliance, workspace, and evidence-gate surfaces.
- Refreshes the README so the repository homepage reflects the current product shape and RC boundary.

## Boundaries

This RC does not claim:

- provider-native bridge readiness
- persistent provider session ownership
- token or cost measurement unless a runtime reports those values directly
- plugin marketplace packaging
- a direct-API patch engine

Token and cost fields remain placeholders by design when providers do not report trustworthy values.
